Laboratório: textos, títulos e links — Site A

Texto canónico e termos alvo

Frase alvo: motor de buscas artesanal. Repetição controlada para testar TF/IDF e BM25: motor de buscas artesanal, motor de buscas artesanal. Este parágrafo inclui variações semânticas: mecanismo de pesquisa, indexador de páginas, crawler, ranking por relevância.

Documento curto

Documento curto com densidade baixa e vocabulário variado. Útil para comparar contra documentos longos.

Documento médio

Inclui subtítulos, listas e âncoras internas. Ver secção de links para comportamento de PageRank.

Documento longo

Texto redundante e sinónimos para avaliar normalização, stemming e stopwords: buscar, pesquisa, procurar, query, consulta.

Variações de links (internos, externos e para o Site B)

  • Link interno com âncora: Metadados
  • Link para Site B (dofollow): Site B — Repositório de Testes
  • Link para Site B com rel=nofollow: Site B (nofollow)
  • Link para Site B com rel=ugc: Site B (ugc)
  • Link externo para diversificar grafo: Exemplo externo

Metadados e sinais on-page

  1. Title/Description: definidos no <head>.
  2. Canonical: aponta para a versão principal deste documento.
  3. OpenGraph: melhora pré-visualizações (não deve afetar ranking textual).
  4. JSON-LD: descreve o site de forma estruturada.

Também podes servir um sitemap.xml e um robots.txt para gerir o crawling.
